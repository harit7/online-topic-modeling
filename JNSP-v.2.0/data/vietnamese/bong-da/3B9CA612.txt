Zidane sẽ là Cầu_thủ hay nhất châu Âu 2003 ?

Hôm_qua , UEFA đã công_bố danh_sách các ứng_viên cho danh_hiệu cầu_thủ hay nhất trong năm .
Zinedine Zidane , Pavel Nedved , Buffon và Van Nistelrooy là những người tham_gia cuộc_đua tranh tới phần_thưởng cao_quý này .

Tiểu_ban kỹ_thuật của UEFA phối_hợp với độc_giả của website   cũng sẽ tiến_hành chọn_lựa những danh_hiệu cá_nhân khác dành cho các_vị_trí thủ_môn , hậu_vệ , tiền_vệ , tiền_đạo .
Kết_quả sẽ được công_bố vào 28/8 , tại Monaco .

Danh_sách đề_cử :
Thủ_môn hay nhất : Gianluigi Buffon ( Juventus ) ; Iker Casillas ( Real Madrid ) ; Francesco Toldo ( Inter Milan ) .

Hậu_vệ hay nhất : Cristian Chivu ( Ajax ) ; Paolo Maldini ( AC Milan ) ; Alessandro Nesta ( AC Milan ) ; Roberto Carlos ( Real Madrid ) ; Lilian Thuram ( Juventus ) .

Tiền_vệ hay nhất : Edgar Davids ( Juventus ) ; Claude Makelele ( Real Madrid ) ; Pavel Nedved ( Juventus ) ; Paul Scholes ( Manchester United ) ; Zinedine Zidane ( Real Madrid ) .

Tiền_đạo hay nhất : Raul Gonzalez ( Real Madrid ) ; Ronaldo ( Real Madrid ) ; Andriy Shevchenko ( AC Milan ) ; Van Nistelrooy ( Manchester United ) ; Christian Vieri ( Inter Milan ) .

Cầu_thủ hay nhất : Zinedine Zidane , Roberto Carlos , Gianluigi Buffon , Pavel Nedved ( Juventus ) ; Ruud van Nistelrooy ( Manchester United ) .
